# Calendar
A Java console application that generates a formatted monthly calendar for any given month and year. It uses modular programming (methods) to calculate total days, identify leap years, and dynamically format the console output.

## Example Output

Enter a year (e.g., 1956): 2026
Enter month in number between 1 and 12: 4
          April 2026
